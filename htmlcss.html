About HTML

Hypertext Markup Language (HTML) is the standard markup language for documents designed to be displayed in web browser. It can be assisted by technologies such as CSS and JavaScript. HTML describes the structure of a web page semantically and originally included cues for the appearance of the document.

About CSS

Cascading Style Cheets (CSS) is a style sheet language used for describing the presentaion of a document written in a HTML. CSS is designed to enable the separation of presentaion and content, including layout, colors, and fonts.
