First appeared : 1995(Netscape Navigator)
ECMA standard : 1997
The newest ECMAScript is ver.9 (2018)
Paradigm: Multi-paradigm: object-oriented, functional

About JavaScript

  1. WHAT IS JAVASCRIPT

    JavaScript was initially created to “make web pages alive” as a browser-only language, but is now used in many other environments as well.
    The programs in this language are called scripts. They can be written right in a web page’s HTML and run automatically as the page loads. Scripts are provided and executed as plain text. They don’t need special preparation or compilation to run.

  2. THREE UNIQUE POINT OF JAVASCRIPT

    • Full integration with HTML/CSS.
    • Simple things are done simply.
    • Support by all major browsers and enabled by default.

    Today, JavaScript has a unique position as the most widely-adopted browser language with full integration with HTML/CSS.
